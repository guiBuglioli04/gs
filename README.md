# Logistica Espacial - Orion Logistics

Aplicativo Android em Jetpack Compose com tres telas e navegacao para o tema de logistica espacial.

## Telas
- Home: apresenta o nome da solucao e resumo do objetivo.
- Envios: lista mockada com filtros e detalhe do envio.
- Rastreio: acompanha o status e atualiza o envio selecionado.

## Como rodar
Use o Android Studio para abrir o projeto e executar em um emulador ou dispositivo.
